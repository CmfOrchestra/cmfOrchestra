JQuery DataTable Row Grouping - Ajax Source

Preamble

DataTables Row Grouping add-in works even if you do not place rows in the table and load them via ajax source.

Initialization code

Initialization code is as in standard case.

$(document).ready( function () {
		$('#example').dataTable({"sAjaxSource": "ajax_source.js"}).rowGrouping();
}
