JQuery DataTable Row Grouping - Ordering groups by different column

Preamble

DataTables Row grouping plugin group rows by cell values and order groups by the value of grouping column. This behaviour can be customized and you can use any other column for ordering groups. As an example you can place additional column with order value used for grouping, and se different column for the actual grouping. In this example rows are grouped by "Rendering engine column but rendering engines are ordered in the following order: Trident, Gecko, Webkit, Presto, Tasman, Misc, KHTML, and other browsers. By default, these groups would be ordered alphabetically by the group name, but in this example is added an additional column that is used for ordering.

Initialization code

In the initialization code you need to define index of the column that will be used for ordering groups iGroupingOrderByColumnIndex in this case set to 0. Due to the fact that grouping column is not a first one it should be set to the value 1. Alternatively you can specify whether the order should be ascending or descending

$(document).ready( function () {
			$(document).ready( function () {
				$('#example').dataTable().rowGrouping({
            							iGroupingColumnIndex: 1,
            							sGroupingColumnSortDirection: "asc",
            							iGroupingOrderByColumnIndex: 0
								});
				
			} );
}
